# Hacker News Post Draft

**Title:** Show HN: RepoMemory – SonarQube for AI Coding Agents (open source)

**Body:**

Your AI coding agent (Claude Code, Cursor, Windsurf) is only as good as the context it receives. But maintaining CLAUDE.md, AGENTS.md, CURSOR.md, and .claudeignore files is a manual chore that most teams neglect — until their agent starts making wrong assumptions.

RepoMemory is an open-source tool that:

• **Audits** your AI context files across 7 quality dimensions (Architecture, Commands, Conventions, Off-limits, Testing, Deployment, Freshness) for 6 AI agents
• **Generates** missing context files — CLAUDE.md, AGENTS.md, CURSOR.md, GEMINI.md, AIDER.md, .claudeignore, slash commands, git hooks
• **Detects drift** — automatically alerts when your code changes but your context files don't update, with risk scoring
• **Fixes everything** in one click — generate + apply all context files from a single button
• **Integrates everywhere** — web app, CLI, MCP server (8 tools), VS Code extension, GitHub Action

It's 100% local-first, works with any OpenAI-compatible API, and runs at zero additional cost.

Built with Next.js 15, TypeScript, shadcn/ui. ~15 core modules, 24 tests, 16 routes.

https://github.com/PraveenJayaprakash-JP/repomemory
